# Codeforces 4A: Watermelon

**Statement:**
One hot summer day Pete and his friend Billy bought a watermelon. They decided to divide the watermelon in such a way that each of the two parts weighs even number of kilos and both parts are non-empty. Pete and Billy are extremely honest, so they want to divide the watermelon in such a way that each of them gets a part of positive weight.

**Input:**
The first and only line contains an integer w (1 ≤ w ≤ 100) — the weight of the watermelon bought by the boys.

**Output:**
Print "YES" if they can divide the watermelon into two parts, each of them weighing even number of kilos; and "NO" otherwise.

**Sample Input 1:**
```
8
```
**Sample Output 1:**
```
YES
```
**Sample Input 2:**
```
3
```
**Sample Output 2:**
```
NO
```
