# Codeforces 1000A: A+B Problem

**Statement:**
Given two integers $a$ and $b$, output their sum.

**Input:**
Two integers $a$ and $b$ ($-1000 \leq a, b \leq 1000$).

**Output:**
Output a single integer, the sum of $a$ and $b$.

**Sample Input 1:**
```
1 2
```
**Sample Output 1:**
```
3
```

**Sample Input 2:**
```
-1000 1000
```
**Sample Output 2:**
```
0
```
